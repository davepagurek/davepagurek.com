This is a sketch for Genuary 2022, which provides a creative coding prompt for each day of January. This is day 16: "Color gradients gone wrong." I'm quite happy with how this one turned out. As you wiggle your mouse around the canvas, it drips paint onto the canvas, which slowly dissolves its colours into the surrounding paint. This uses shader feedback: each frame, it samples colours from the previous frame, but with a little bit of distortion to get the colours to spread around.
