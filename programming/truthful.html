In both our Math 135 (Algebra) and ECE 124 (Digital Circuits) courses this term, we've had to work with boolean algebra. This can get tricky and often to check our work, we make truth tables. This is a tool to enter boolean expressions and generate truth tables.

Expressions

Create an algebraic expression like this: var expression = Truthful.createExpression("a*b+!(c*d)"); To get the result of an expression: var x=2; var result = expression.result({ "a": 1, "b": 0, "c": 1, "d": 1 }); If a variable's value is not specified, it defaults to zero. Expressions can include * for AND. + for OR, and ! for NOT.
