Mindflow is a simple but effective flowchart and diagramming application for Windows, Mac and Linux through Adobe Air. It is made to create a visual representation of your thoughts which you can then easily use in future projects.

 

...or, get Mindflow for Blackberry Playbook

 

Features

  • Easy multi-branched flowchart creation
  • Automatic conversion to outline
  • Export flowchart as JPG
  • Export outline as HTML list
  • Realtime spell checking
  • ...and more!
